## Быстрый старт (на русском)

1. Установите зависимости (если ещё не установили):

```bash
npm install
```

2. Запустите дев-сервер:

```bash
npm run dev
```

3. Откройте http://localhost:3000 — вы увидите тёмную страницу с интерактивным mixed-графиком.

Файлы, которые созданы для тестового задания:

- `src/app/data.ts` — интерфейс `ChartPoint` и массив `data` с тестовыми значениями (10–15 точек).
- `src/app/components/AnalyticsChart.tsx` — клиентский компонент с `ComposedChart` (Area, Line(s), Bar) и shared tooltip.
- `src/app/page.tsx` — страница, центрирующая график на премиальном тёмном фоне.

Как подставить свои данные

1. Откройте файл `src/app/data.ts`.
2. Замените массив `data` на свои 4 последовательности. Каждый объект должен быть формата:

```ts
{
  name: string; // метка/дата
  areaValue: number;
  splineValue: number;
  lineValue: number;
  barValue: number;
}
```

3. Фронтенд автоматически отобразит новые значения при перезагрузке (или в режиме dev — изменения подхватятся автоматически).

Примечания

- Проект использует `recharts`. Если вы ещё не устанавливали, выполните `npm install recharts`.
- Компонент оформлен в тёмном премиальном стиле с неоновыми акцентами — цвета легко править в `src/app/components/AnalyticsChart.tsx`.
